# v2.7.0 - AI Advisory and Safe Resource Exhaustion Demo

## Summary

v2.7.0 adds deterministic AI advisory capabilities and safe HTTP/2 Resource Exhaustion demo support while preserving rule-based detection authority. The release expands analyst-facing context through AI Analyst Brief, Evidence Gap Analyzer, defensive RAG knowledge routing, and a synthetic Resource Exhaustion scenario without increasing AI authority over Risk Level, Decision, or any operational enforcement path.

## Highlights

- Deterministic AI Analyst Brief backend.
- AI Analyst Brief UI panel.
- Evidence Gap Analyzer backend and UI.
- Resource Exhaustion / HTTP/2 DoS / CVE knowledge pack.
- Expanded RAG routing for HTTP/2 / Resource Exhaustion / CVE questions.
- RAG answer safety framing.
- CVE vs CVSS terminology normalization.
- Safe synthetic HTTP/2 Resource Exhaustion demo scenario.
- Manual smoke report.
- Release gate passed.

## Safety Boundaries

- Detector remains rule-based.
- Risk Level and Decision remain deterministic.
- `BLOCK`, `MONITOR`, and `ALLOW` remain simulated.
- RAG, LLM-assisted explanation, AI Analyst Brief, and Evidence Gap Analyzer are advisory only.
- No real firewall, WAF, EDR, account, cloud, SIEM, SOAR, or monitoring action is performed.
- No exploit, proof-of-concept, or traffic generation is provided.
- Human review is required.
