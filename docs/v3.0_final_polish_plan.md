## Safety Boundary (unchanged)

- Rule-Based Detector remains the detection authority.
- Risk Level / Decision remain deterministic.
- BLOCK / MONITOR / ALLOW are simulated decisions only.
- The Evidence-Grounded AI Brief is advisory-only.
- RAG / Similar Cases / Graph context cannot override the official verdict.
- Similar Cases are not proof of current compromise.
- Graph context is not a detection source.
- Human review is required.
- No real firewall / WAF / EDR / account / cloud / SIEM / SOAR enforcement is performed.
- This is an academic / portfolio-grade blue-team triage prototype, not a production
  IDS/IPS and not a red-team tool. No live LLM client is wired.

## Non-Goals

- No new major features; no live LLM client.
- No change to detector, risk scoring, decision policy, RAG retrieval semantics,
  graph fact builder, or ToolPolicy.
- No real enforcement; no exploit / PoC / traffic generation.
- No GitHub Release, tag, merge, reset, or rebase as part of this polish pass.
